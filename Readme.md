# 🧮 MATH MASTER - คิดเลขเร็ว

เกมฝึกทักษะการคำนวณคณิตศาสตร์พื้นฐาน (+, -, \*, /) แบบจับเวลา โดยมีระบบตัวเลือก (Multiple Choice) ตอบถูกได้คะแนนและรีเซ็ตเวลา ตอบผิดหรือเวลาหมด = Game Over!

## 🚀 Features

- **Dynamic Questioning:** สุ่มโจทย์ไม่ซ้ำกัน
- **Responsive UI:** หน้าจอสวยงาม รองรับทั้งมือถือและคอมพิวเตอร์
- **Modern Design:** ใช้ Glassmorphism และ Google Fonts (Kanit)
- **Immediate Feedback:** ระบบเช็คคำตอบทันทีพร้อม Animation ตอนแพ้

## 🛠️ Tech Stack

- **HTML5** - โครงสร้างเว็บ
- **CSS3** - ตกแต่งหน้าจอ (Custom Animations & Gradients)
- **JavaScript (ES6)** - ระบบ Logic เกม, Timer และ Event Handling

## 🎮 How to Play

1. กดปุ่ม **"เริ่มเกม"**
2. เลือกคำตอบที่ถูกต้องภายในเวลาที่กำหนด
3. หากตอบถูก เวลาจะถูกรีเซ็ตกลับมา และคะแนนจะเพิ่มขึ้น
4. หากตอบผิดหรือเวลาเหลือ 0 เกมจะจบทันที
